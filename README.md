#  智能衣架

## 采购清单

1. 温湿度传感器：DHT11
2. 拉力传感器：LFS-02
3. Wi-Fi模块：ESP8266
4. stm32芯片：F103
5. 电源模块：18650电池+18650电池电源转换模块
